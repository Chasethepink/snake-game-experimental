# 🐍 贪食蛇游戏

一个功能丰富的现代化贪食蛇游戏，支持多种难度、背景音乐和音效。

## 🌐 在线演示

[点击这里在线体验](https://chasethepink.github.io/snake-game-experimental/)

## 🎮 游戏特点

- 🎨 现代化渐变UI设计
- 🐍 流畅的蛇移动动画
- 🍎 随机生成食物
- 📊 实时分数和最高分记录（本地存储）
- ⚡ 难度递增系统
- 🎵 背景音乐支持
- 🥁 吃食物时的鼓点音效
- 🔇 静音开关

## 📁 文件说明

```
贪食蛇/
├── index.html              # 游戏主文件
├── background-music.mp3    # 背景音乐文件（需要自行添加）
└── README.md              # 说明文档
```

## 🚀 如何开始

### 方法一：直接运行（无音乐）
1. 双击 `index.html` 文件
2. 点击"开始游戏"按钮即可开始玩

### 方法二：添加背景音乐
1. 从 YouTube 下载音频：https://youtu.be/1h1QukxXXlw
   - 使用在线工具如：y2mate.com, yt1s.com
   - 或使用浏览器插件/桌面软件下载
2. 将音频文件转换为 MP3 格式
3. 重命名为 `background-music.mp3`
4. 放在与 `index.html` 同一文件夹中
5. 双击 `index.html` 开始游戏

## 🎯 游戏玩法

### 基本操作
- ⬆️ 方向键：控制蛇的移动
- 🎮 开始游戏：点击"开始游戏"按钮
- ⏸️ 暂停/继续：点击"暂停/继续"按钮

### 游戏规则
1. 使用方向键控制蛇的移动方向
2. 吃到红色食物得 10 分，蛇会变长
3. 避免撞到墙壁和自己的身体
4. 随着分数增加，蛇的移动速度会变快

### 难度设置
- **简单**：初始速度 250ms，速度递增缓慢
- **普通**：初始速度 150ms，速度递增适中（默认）
- **困难**：初始速度 100ms，速度递增较快
- **专家**：初始速度 70ms，速度递增非常快

## 🎵 音效说明

### 背景音乐
- 循环播放 BLACKPINK 的音乐
- 音量默认为 30%
- 可通过静音按钮开关

### 游戏音效
- **吃食物**：三层鼓点音效（Kick + Snare + Hi-hat）
- **游戏结束**：低沉的结束音

## ⚙️ 技术特性

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- Web Audio API 音效系统
- LocalStorage 最高分存储
- 响应式设计

## 📊 游戏参数

- 画布大小：506 x 506 像素
- 网格大小：22 像素
- 网格总数：23 x 23
- 初始蛇长度：1 格

## 🎨 主题颜色

- 背景渐变：紫色系 (#667eea → #764ba2)
- 蛇头：绿色 (#4CAF50)
- 蛇身：浅绿色 (#8BC34A)
- 食物：红色渐变 (#ff6b6b → #ee5a6f)

## 💾 本地存储

游戏会自动保存你的最高分到浏览器本地存储，即使关闭浏览器后再打开，最高分记录依然保留。

## 🔧 自定义修改

如果你想修改游戏参数，可以编辑 `index.html` 文件中的以下变量：

```javascript
// 难度设置
const difficultySettings = {
    easy: { speed: 250, speedIncrease: 1 },
    normal: { speed: 150, speedIncrease: 2 },
    hard: { speed: 100, speedIncrease: 3 },
    expert: { speed: 70, speedIncrease: 4 }
};

// 音乐音量
bgMusic.volume = 0.3; // 0.0 到 1.0 之间

// 网格尺寸
const gridSize = 22;
const tileCount = 23;
```
